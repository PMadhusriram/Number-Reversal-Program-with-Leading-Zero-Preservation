# Number-Reversal-Program-with-Leading-Zero-Preservation
This Python program takes an integer input from the user, reverses the number using mathematical operations, and ensures that any leading zeros lost during the reversal process are restored in the final output.
What the Program Does:

Accepts an integer input from the user.

Counts the number of digits in the original input.

Reverses the number using arithmetic operations:

Extracts the last digit using modulus (% 10)

Builds the reversed number using multiplication and addition

Removes the last digit using floor division (// 10)

Counts the digits of the reversed number.

Calculates the difference in digit count between the original and reversed numbers.

Adds back any lost leading zeros to maintain the original digit length.

Prints the final reversed number with correct formatting.
